Php7 MVC básico

Sintaxe básica de padrão Php7 MVC

Introdução

Padrão de arquitetura de software que divide a aplicação em três camadas: Model, View e Controller.

  • Model: Faz a conexão, leitura, escrita e validação dos dados. Nela são implementadas as regras de negócio: definições de como a empresa faz negócios. É a declaração de como a empresa trabalha, os processos para execução do serviço, ou entrega do produto que a mesma trabalha. Modificações nos dados.
  • Controller: Lida com as requisições do usuário. Gerenciando as ações realizadas, identifica qual Model e qual View utilizar para que a ação seja completada. Modificações nas requisições.
  • View: Faz a interação com o usuário (front-end). Nela são apresentarão-se os dados ao usuário, que podem ser entregues em vários formatos, como HTML, XML, documentos, vídeos, fotos, músicas etc. Manutenções no layout.

MODEL - CONTROLLER - VIEW - (usuário)

Vantagens
  • Reusabilidade de códigos
  • Desenvolvimento Rápido
  • Fácil manutenção
Exemplo de funcionamento básico:
  1. View: Usuário acessa tela com formulário de login, apresentada pela View, que encaminha os dados informados ao controller.
  2. Controller: Recebe dados e encaminha para a Model correta validar.
  3. Model: Valida dados e retorna a resposta de login.
  4. Controller: Pega a validação feita pela Model e encaminha à view correta, para que eles sejam exibidos ao usuário.
  5. View: Exibe tela resultado ao usuário (logou ou não logou).

index.php


<?php
    include '/mvcexemplo/Controller/Controller.php';
    include '/mvcexemplo/View/View.php';
    include '/mvcexemplo/Model/Model.php';

    $acao = $_GET['acao'];

    if(isset($acao) && $acao == 'login'){
        $controller = new Controller();
        $controller->login();
    } else {
        $view = new View();
        $view->login(); 
    }
?>

view.php


<?php
    class View {
        public function login($validacao) { 
            if(!isset($validacao)) {
                echo ' 
                    <h3>Login</h3>
                    <form action='index.php' method ='post'>
                        <input type='text' placeholder='Usuário' name='usuario'>
                        <input type='password' placeholder='Senha' name='senha'>
                        <input type='submit' value='Entrar'>
                    </form>
                ';
            } else {echo '<h3>'.$validacao.'</h3>';}
        }
    }
?>

controller.php


<?php
    include '/../Model/Model.php';
    class Controller {
        public function login() {
            $user = $_POST['usuario'];
            $pass = $_POST['senha'];
            $model = new Model();
            $validacao = $model->validaDados($user,$pass);
            $view = new View();
            $view->login($validacao); 
        }
    }
?>

model.php


<?php
    class Model {
        public function validaDados($user,$pass) {
            if(($user!="admin") and ($pass!="adm")) {return 'Login incorreto'; }
            else {return 'Logado com sucesso';}
        }
    }
?>
