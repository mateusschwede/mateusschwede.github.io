Markdown: Linguagem de documentação web

Conceitos e sintaxe da linguagem Markdown

O que é?

Markdown é uma linguagem de marcação, convertida pra Html, muito utilizada para escrever documentação web. Devido à sintaxe 'populosa' do Html e sua difícil interpretação quando aplicada à escrita de documentação web, fora necessária a criação de uma linguagem de marcação com sintaxe mais limpa e focada ao texto, dando origem à Markdown. Você encontrará Markdown principalmente nos arquivos 'readme.md', que são utilizados como um breve manual para determinado software/projeto. Toda a documentação online da Microsoft também é escrita com auxílio da linguagem. Arquivos Markdown possuem extensão '.md'. Você pode testar seus códigos no editor online Dillinger. Mais informações e documentação, o site oficial da linguagem encontra-se no link 'www.markdownguide.org'. Encontre também um resumão oficial no link 'www.markdownguide.org/cheat-sheet'.
