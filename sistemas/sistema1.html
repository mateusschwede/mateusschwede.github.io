EChamado

Software gestor de chamados para suporte técnico

Pré-requisitos

Necessário ter o Xampp executando. 1º execute o banco.sql no Phpmyadmin, para então abrir o projeto no localhost. Entre com o login Admin(nome:admin senha:111) para realizar as demais operações. Para mais detalhes, acesse o Readme.md no Github.

Acesse no Github >

Como funciona

O sistema inicia com um admin, que possui como funções gerenciar o cadastro de técnicos, máquinas e clientes. Isso, além da visualização de relatórios de chamados por data, máquina, cliente ou técnico. Também há o técnico, com a função de receber os chamados, analisá-los e finalizá-los, além de conferir seu próprio histórico de chamados finalizados por data. Por fim, tem-se o cliente, com função de adicionar novos chamados, além de cancelá-los, acompanhá-los, por meio de mensagens no sistema, o andamento detalhado de seu chamado e finalização. Isso, além de seu próprio histórico de chamados por data.



Tecnologias

  • Php7.4 (Pdo)
  • MySQL (MariaDB10)
  • Bootstrap4.5

Algun erro?

OBS: Remova o underline do &_times para o código funcionar

Caso tenha encontrado algum erro bobo, como falta de tag ou algo assim, saiba que ao realizar esse post tive que trocar a abertura de tag por &_lt(sem o underline), e o fechamento de tag por &_gt(sem o underline). Certifique-se de 1º executar o .sql, para então acessar o sistema. Verifique se todos os nomes encontram-se idênticos aos informados, e que todos encontram-se dentro da mesma pasta. Saiba que a versão do Php deve ser acima da 7, assim como a do Bootstrap deve ser 4.5, e o MySQL também atualizado. Se o erro persistir, envie-me um email, mateus1908.schwede@gmail.com, informando a mensagem do erro e a situação em que o mesmo se encontra, será uma honra concertá-lo!

    ativo BOOLEAN NOT NULL DEFAULT 1
) CHARSET=utf8;

    id INTEGER AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(50) NOT NULL,
    senha VARCHAR(5) NOT NULL,
    ativo BOOLEAN NOT NULL DEFAULT 1,
    ipMaquina VARCHAR(30)
) CHARSET=utf8;

CREATE TABLE tecnico (
    id INTEGER AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(50) NOT NULL,
    senha VARCHAR(5) NOT NULL,
    ativo BOOLEAN NOT NULL DEFAULT 1
) CHARSET=utf8;

CREATE TABLE chamado (
    id INTEGER AUTO_INCREMENT PRIMARY KEY,
    dthrCadastro DATETIME NOT NULL DEFAULT now(),
    descricao VARCHAR(500) NOT NULL,
    tipo VARCHAR(10) NOT NULL,
    situacao VARCHAR(10) NOT NULL DEFAULT "pendente",
    ipMaquina VARCHAR(30),
    idCliente INTEGER NOT NULL,
    idTecnico INTEGER,
    dthrAnalise DATETIME,
    dthrFinalizado DATETIME
) CHARSET=utf8;

CREATE TABLE notificacao (
    id INTEGER AUTO_INCREMENT PRIMARY KEY,
    conteudo VARCHAR(200) NOT NULL,
    tipo VARCHAR(10) NOT NULL,
    idCliente INTEGER NOT NULL
) CHARSET=utf8;
